Detailed Description

DHCP Client Application. When connected to the uIP stack, this will retrieve IP configuration settings from the DHCP server on the network.

Fills the DHCP packet response with the appropriate BOOTP header for DHCP. This fills out all the required fields, leaving only the additional DHCP options to be added to the packet before it is sent to the DHCP server.
