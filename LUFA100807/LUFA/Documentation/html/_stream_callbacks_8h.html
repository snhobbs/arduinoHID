Detailed Description

This file contains definitions for the creation of optional callback routines which can be passed to the endpoint and/or pipe stream APIs, to abort the transfer currently in progress when a condition is met.

Note
This file should not be included directly. It is automatically included as needed by the USB driver dispatch header located in LUFA/Drivers/USB/USB.h.
