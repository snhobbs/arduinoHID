Detailed Description

RNDIS command handler functions. This handles RNDIS commands according to the Microsoft RNDIS specification, creating a USB Ethernet network adapter.

uint8_t RNDISMessageBuffer[sizeof(AdapterSupportedOIDList)+sizeof(RNDIS_Query_Complete_t)]

Buffer for RNDIS messages (as distinct from Ethernet frames sent through the adapter. This must be big enough to hold the entire Supported OID list, plus the response header. The buffer is half-duplex, and is written to as it is read to save on SRAM - for this reason, care must be taken when constructing RNDIS responses that unread data is not overwritten when writing in responses.
